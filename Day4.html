In HTML, CSS -> imagine every element as a box. Box can have 3 things(BOX MODEL):

  • Border
  • Padding
  • Margin

Border-width -> can't be '0'. If its 0, then border is not visible.
Boder-style -> must be specified, to view the border.(solid,dotted,dashed)
Instead of writing Border-style,Border-width,Border-color, we can use simply "border" to specify all those 3.
border: width style color ;
Border-radius -> roundness of the corner of the border!!
Padding
Paddng-> is a spacing b/w border and content in the border.
To give Padding in all directions:
Using -> Padding-top,padding-right,padding-bottom,padding-left.
or
Instead of listing above, it is possible to use ->
padding: top right bottom left ; (clock-wise)
padding-inline -> padding in left & right directions only!....
padding-block -> padding in top and bottom only.!.....
To give padding same in all directions, we use padding: __px ;
text-align -> align the text in (left,right, center, justify(fill up the space)).
margin -> space around border(box)
margin: __px;
margin:auto; (auto -> provides margin equally in all directions.)
margin-inline -> spacing around left and right of the border.( if margin-inline -> auto, then box aligns at the center (equal amount of margin at left & right.)) margin-block -> spacing around top and bottom of the border.
